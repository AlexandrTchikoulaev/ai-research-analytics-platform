Ajuda

Manual de instruções — encontre aqui resposta para as suas dúvidas
Esta aplicação serve para gerir relatórios e os dados que estão dentro desses relatórios. Pode adicionar documentos PDF, associar ficheiros de dados a esses documentos, e depois visualizar essa informação em gráficos e tabelas. Se tiver alguma dúvida sobre como usar qualquer parte da aplicação, encontra a resposta abaixo.
Relatórios

Vá à página Adicionar e certifique-se que está no separador Relatórios.

Tem dois modos à sua escolha:

  • Formulário — preencha os campos um a um e clique "Adicionar Relatório". Use este modo quando quiser adicionar um relatório de cada vez.
  • JSON — cole ou carregue um ficheiro com vários relatórios ao mesmo tempo. Use este modo quando tiver muitos relatórios para inserir de uma vez.

Para mudar de modo, clique no botão "ir para JSON" ou "ir para Formulário" no canto superior direito do formulário.

  • Título — o nome do ficheiro PDF (ex: relatorio_2024.pdf). Serve para identificar o relatório.
  • Link — o endereço do PDF na internet. Se o documento não estiver online, pode fazer upload diretamente (ver pergunta sobre Link vs. Ficheiro).
  • Fonte — a organização ou entidade que publicou o relatório (ex: IMF, Banco Mundial, INE).
  • Data de publicação — quando o relatório foi publicado.
  • Área temática — a categoria do relatório (Economia, Saúde, Educação, etc.).
  • Estado — se é um relatório Anual, Bienal, Pontual ou Descontinuado.
  • Palavras-chave — termos que descrevem o conteúdo, separados por vírgula (ex: inflação, PIB).
  • Resumo — uma breve descrição do relatório.
O título e a data de publicação são obrigatórios. Os restantes campos são opcionais mas ajudam a organizar e encontrar os relatórios mais facilmente.

No modo JSON, escreva ou cole uma lista de relatórios no formato indicado. Cada relatório é um bloco entre chavetas {}, e todos os blocos ficam dentro de parênteses retos [].

Os campos são os mesmos do formulário: título, link, fonte, data de publicação, etc. Pode também carregar um ficheiro .json diretamente em vez de colar o texto.

Se um dos relatórios tiver algum problema (ex: título duplicado), só esse falha — os restantes são inseridos normalmente.
Dados

Vá à página Adicionar e mude para o separador Dados.

Os dados são ficheiros com informação numérica (tabelas de estatísticas, valores por país, etc.) que pertencem a um relatório. Para os adicionar:

  • Escolha a qual relatório estes dados pertencem (clique no campo para ver a lista).
  • Indique onde estão os dados — um endereço de internet (Link) ou um ficheiro do seu computador (Ficheiro).
  • Escolha a função adequada para que o sistema saiba como ler o ficheiro.
  • Clique "Adicionar Ficheiro".

Tal como nos relatórios, também existe um modo JSON para inserir vários registos de uma vez.

A função diz ao sistema como deve ler e interpretar o ficheiro de dados — qual é a estrutura, onde estão os valores, etc. Cada tipo de ficheiro pode ter o seu formato específico, por isso é importante escolher a função certa.

Se não sabe qual usar, pode clicar no botão "Gerar": o sistema analisa o ficheiro e tenta criar uma função adequada automaticamente. Depois de gerada, ela fica disponível para seleção.

Se o sistema não conseguir gerar uma função automaticamente, ou se os dados não ficarem corretos no Dashboard, pode ser necessário pedir ajuda a alguém mais técnico para criar uma função manualmente.

Sim. Em modo Ficheiro, pode selecionar vários ficheiros de uma vez, e também pode escolher várias funções ao mesmo tempo. O sistema cria uma entrada para cada combinação de ficheiro e função.

Por exemplo: se carregar 2 ficheiros e selecionar 3 funções, o sistema cria automaticamente 6 entradas. Uma pré-visualização aparece antes de confirmar, para que possa ver o que vai ser criado.

Use esta funcionalidade quando precisar de extrair tipos diferentes de informação dos mesmos ficheiros.
Link e Ficheiro

Ao adicionar um relatório ou dados, tem sempre a opção de usar Link ou fazer upload de um Ficheiro.

  • Link — o documento está disponível na internet. Cole o endereço e o sistema vai buscá-lo automaticamente quando processar. Use esta opção quando o ficheiro está publicado online.
  • Ficheiro — faça upload do ficheiro diretamente para o sistema. O ficheiro fica guardado internamente. Use esta opção quando só tem o documento no seu computador e ele não está disponível online.
Para relatórios, apenas são aceites ficheiros PDF. Para dados, são aceites vários formatos: JSON, CSV, Excel, XML e ZIP.
Registos

Quando um relatório ou ficheiro de dados já foi processado pelo sistema e os seus dados estão guardados, ele fica bloqueado e não pode ser apagado diretamente.

Isto é uma proteção para garantir que não se perdem dados que já estão integrados no sistema.

Se precisar de corrigir algo num registo bloqueado, use o botão "Editar". Ao guardar as alterações, o sistema vai tentar reprocessar esse registo automaticamente na próxima vez que correr.

Ao editar qualquer campo e guardar, o sistema marca o registo como "novo" — como se tivesse sido inserido agora. Isso faz com que seja reprocessado automaticamente na próxima vez que o sistema correr.

Os erros anteriores associados ao registo são também limpos, dando-lhe uma nova oportunidade de ser processado com sucesso.

Painel de Controlo

O Painel de Controlo mostra os registos que tiveram problemas durante o processamento automático do sistema.

  • Separador Relatórios — PDFs que o sistema tentou ler mas encontrou algum problema.
  • Separador Dados — ficheiros de dados que não foram processados corretamente.

Se aparecer um registo aqui, significa que algo correu mal. Pode clicar no registo para ver a mensagem de erro e editá-lo para tentar corrigir o problema. Depois de editado, o sistema tentará reprocessá-lo automaticamente.

Se os erros forem muitos e já não forem relevantes, pode usar o botão "Apagar tudo" para limpar a lista.
Dashboard e Chat

A página Dashboards mostra os dados que já foram processados e estão prontos para consulta.

Selecione um indicador (o tipo de dado que quer ver, por exemplo: taxa de inflação, PIB per capita) e um ano. O sistema mostra um gráfico com os valores por país ou região.

Pode também filtrar por um relatório específico para ver apenas os dados desse relatório, ou ver todos os relatórios combinados.

Só aparecem no Dashboard os dados que já passaram pelo processamento completo. Se acabou de inserir dados, pode demorar até ao próximo ciclo de processamento para aparecerem.

Na página Chat pode fazer perguntas em linguagem normal e o sistema tenta responder com base nos relatórios e dados que tem guardados.

Por exemplo, pode perguntar:

  • "Qual é a taxa de inflação em Portugal em 2023?"
  • "O que diz o relatório do FMI sobre crescimento económico?"
  • "Que países têm o PIB mais elevado?"

O Chat funciona em dois modos: um para perguntas sobre o conteúdo dos PDFs, e outro para perguntas sobre os valores numéricos dos dados.
